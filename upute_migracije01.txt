Prvo trebamo upaliti XAMPP i pokrenuti Apache i MySQL

Apache nam treba da bi pokrenuti phpMyAdmin

Link http://localhost/phpmyadmin

Ako traži login, defualtni podaci su
Username: root
Lozinka: nema lozinke

Idemo napraviti našu praznu bazu podataka u koju ćemo tablice
dodavati preko SQLAlchemy.

Baza se zove "aup1" i nema tablica.

Sada trebamo instalirati SQLAlchemy, Flask-migrate i PyMySQL
pip install flask-sqlalchemy flask-migrate pymysql

Da bi koristili virtualno okruženje projekta (.venv)

sve naredbe poput PIP moraju ići nakon "python -m"
Pa je konačna naredba

python -m pip install flask-sqlalchemy flask-migrate pymysql

Vidimo da je uspješno prošlo, idemo provjeriti requirements.txt

Tu se ne dodaju packagei automatski, pa idemo
python -m pip freeze > requirements.txt

Sad je u redu. Aplikaciju pokrećemo sa
python main.py

a kako smo definirali debug=True u programskom kodu tako će se automatski
naše promjene osvježavati.

----

Prelazimo na drugi dio, idemo napraviti dvije datoteke
extensions.py i models.py

Označimo nove datoteke i idemo Ctrl + Alt + A - dodaju se na git.

---

Zbog jednostavnosti ćemo napraviti app.py umjesto main.py
pa ćemo na idućim vježbama povezati rute sa ovim što danas radimo.

flask db init
flask db migrate -m "napravljena tablica profesori"

za sad još nije ništa napravljeno osim "alembic" tablice koja će pratiti promjene u našim migracijama

da bi se stvarno napravilo, pokrenut ćemo
flask db upgrade

--

idemo napraviti tablicu "kolegiji"
flask db migrate -m "napravljena tablica kolegiji"
flask db upgrade


# idemo sada povezati ove dvije tablice sa stranim ključem i definirat ćemo relacije između njih

# nećemo praviti nove modele nego uređujemo postojeće, a alembic sam prepoznaje promjene.
# vraćamo se na models.py

flask db migrate -m "Povezan Profesor i Kolegij"
flask db upgrade

idemo sada dodati kolegij kroz programski kod, odnosno kroz rutu za dodavanje
nastavit ćemo raditi u app.py
